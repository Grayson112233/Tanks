Tanks, a pygame by Grayson Pike, 2014

  - Toggle fullscreen with enter
  - Move tank with WASD
  - Move tank turret left/right with arrow keys
  - Fire gun with space bar
  - Exit game with escape
  
* Requires Python 3 and pygame installation to run
